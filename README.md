# Description #

## Autofill Events from The Game Planning to Google Calendar ##

This extension automates the process of retrieving scheduled events from planning.thegame-france.com and adds them to your Google Calendar, allowing for easier event management and scheduling.

### Features ###
- Automated Data Extraction: Fetch events directly from the planning.thegame-france.com website.
- Google Calendar Integration: Automatically populate events in Google Calendar based on the extracted data.
- Customization: Configurable to retrieve specific event details like event name, date, time, and location.

# Usage #

1.
    Open **The Game Planning** Site:
        Navigate to The Game Planning.
        Ensure you’re logged in.
2.
    Trigger the Autofill:
        The extension script will automatically detect and extract relevant event information, such as the date, time, and title.
3.
    Confirm Google Calendar Entry:
        The extracted event details are prepared and redirected to Google Calendar’s URL format.
        Review and confirm the details in Google Calendar to add the event.
